SESSION MANAGEMENT

用 Session 同時處理多件事

Session 是大閘蟹的核心概念，讓你像開多個瀏覽器分頁一樣，讓 AI 同時處理不同任務。

?

什麼是 Session？

每個 Session 是一個獨立的對話上下文。

Session A
📈 股票監控
持續監控 TSLA、2330 即時報價
Active
Session B
💬 日常問答
翻譯、搜尋、一般對話
Active
Session C
📊 資料整理
長時間資料分析任務（執行中）
Running
🔀
互不干擾
各 Session 完全獨立
⚡
同時並行
多任務同步執行
🧠
獨立記憶
各自有專屬上下文

如何使用 Session

1

建立新 Session

/session new 股票監控

→ 建立一個名為「股票監控」的 Session 

/session new 日常助理

→ 再建立一個「日常助理」Session

2

切換 Session

/session switch 股票監控

→ 切換到「股票監控」Session，之後的對話都在這個 Session 進行 

/session list

→ 列出所有 Session 及其狀態

3

刪除 Session

/session delete 股票監控

⚠️ 刪除 Session 會同時清除該 Session 的所有對話記憶，無法復原。重要的 Session 請謹慎刪除。

實際應用範例

💰
財經監控 + 日常助理分離
Session "財經"
設定每日財經早報排程、股票提醒
Session "助理"
日常問答、翻譯、摘要任務
✅ 兩者互不影響，財經排程不會被日常對話打斷
⚡
長任務 + 即時查詢
Session "長任務"
整理大量資料、分析報告（可能需要數分鐘）
Session "即時"
同時進行其他查詢，不需等待長任務完成
✅ 不再卡在等待，多工並行
📁
多專案管理
Session "專案A"
追蹤 A 專案相關資料、待辦
Session "專案B"
追蹤 B 專案，各自有獨立記憶
✅ 就像有多個專屬助理，各司其職

💡 Session 使用技巧

  • • Session 名稱建議有意義（用途/專案名），方便記憶
  • • 重要 Session 不要隨意刪除，記憶會一起消失
  • • 切換 Session 前可以先 /session list 確認有哪些
  • • 排程任務建議放在獨立 Session，避免被日常對話干擾
SKILL MANAGEMENT

透過 ClaWHub 安裝技能

Skills 是大閘蟹的插件系統，讓你無限擴充 AI 的能力。

🏪

什麼是 ClaWHub？

ClaWHub 是大閘蟹的技能市集，類似 App Store。你可以在 Telegram 對話中直接安裝、移除、更新技能，無需登入任何後台。

三種安裝方式

1

直接告訴 AI（最簡單）

最直觀的方式：直接用自然語言告訴大閘蟹你想要什麼，它會自動幫你執行。

你： 幫我安裝 stock-info-explorer 技能
🦀： 好的，正在為你安裝 stock-info-explorer... 安裝完成！現在你可以問我股票價格了。

ℹ️ AI 會自動執行安裝並確認結果。如果有錯誤，也會自動嘗試修復。

2

直接在伺服器執行指令

進階用戶可以透過 SSH 直接在伺服器執行 clawhub 指令。

bash 
# 安裝單一技能
clawhub install SKILL_NAME

# 安裝需要強制安裝的技能（VirusTotal 誤判）
clawhub install --force SKILL_NAME

# 移除技能
clawhub remove SKILL_NAME

# 更新所有技能
clawhub update

# 列出已安裝技能
clawhub list

# 搜尋可用技能
clawhub search KEYWORD

💡 部分技能需要加上 --force 參數，這是因為 VirusTotal 對某些技能有誤判，並非真的有安全問題。

3

讓 AI 自建技能（skill-creator）

安裝 skill-creator 技能後，你可以告訴 AI 你想要什麼功能，它會自動開發、測試並安裝。

你： 幫我做一個每天爬取 PTT 科技板熱帖、整理 5 條重點的技能
🦀： 好的，我來幫你建立這個技能，正在分析 PTT 結構...
[AI 自動寫程式、測試、安裝]
🦀： 技能「ptt-tech-summary」已安裝，每天會自動執行，並將重點摘要傳送給你。

✨ skill-creator 讓大閘蟹的能力真正無限。只要你想得到，都可以讓它幫你開發。

RECOMMENDED SKILLS

推薦技能清單

以下是精選的實用技能，覆蓋日常使用的核心需求

🔍

ddg-web-search

搜尋

DuckDuckGo 即時搜尋，不需要任何 API Key。詢問最新資訊時自動觸發。 

clawhub install ddg-web-search
✅ 免費 ✅ 無需 API Key ⭐ 預裝推薦
📈

stock-info-explorer

金融

美股、台股即時報價。支援 TSLA、台積電 (2330)、ETF 等，基於 Yahoo Finance。 

clawhub install --force stock-info-explorer
✅ 免費 ✅ 即時數據 ⚠️ 需 uv
₿

crypto-market-data

金融

主流加密貨幣即時行情，BTC、ETH、SOL 等，含 24h 漲跌幅。 

clawhub install --force crypto-market-data
✅ 免費 ✅ 即時數據
📰

news-summary

資訊

指定主題的今日新聞摘要，AI 自動搜尋並整理成重點列表。 

clawhub install --force news-summary
✅ 支援任意主題 ✅ 繁中支援
🎬

youtube-transcript

工具

貼入 YouTube 網址，自動擷取字幕並整理重點，省時省力。 

clawhub install youtube-transcript
✅ 免費 ✅ 支援多語言字幕
🌐

translate

語言

多語言互譯，中英日韓等，AI 驅動比傳統翻譯更自然流暢。 

clawhub install translate
✅ 免費 ✅ AI 驅動
📄

summarize

工具

長文摘要整理，貼入文章、報告、合約，快速提取重點。 

clawhub install summarize
✅ 免費 ✅ 支援超長文本
🌤️

weather

生活

全球城市即時天氣查詢，基於 Open-Meteo，不需要任何 API Key。 

clawhub install weather
✅ 免費 ✅ 無需 API Key ⭐ 預裝推薦
🛠️

skill-creator

進階

讓 AI 幫你自動開發新技能。告訴 AI 你的需求，它寫程式、測試、安裝，一條龍完成。 

clawhub install --force skill-creator
✅ 無限擴充 ✅ 自然語言操作
一次安裝所有推薦技能

clawhub install ddg-web-search weather summarize translate youtube-transcript
clawhub install --force skill-creator stock-info-explorer news-summary crypto-market-data
