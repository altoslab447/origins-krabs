🔑 STEP 1

取得 AI API Key

大閘蟹支援多種 AI 模型，你需要一個 API Key 才能啟動。建議新手從 Gemini 開始（有免費額度）。

🌀
Google Gemini
推薦新手・有免費額度
1
前往 aistudio.google.com
2
登入 Google 帳號
3
點「Get API Key」→「Create API Key」
4
複製 Key（格式：AIza...）
🐬
DeepSeek
超高 CP 值・費用不到 OpenAI 的 1/10
1
前往 platform.deepseek.com/api_keys
2
點「Create API Key」
3
複製 Key（格式：sk-...）
⚡
OpenAI GPT
需先儲值才能使用
1
前往 platform.openai.com/api-keys
2
點「Create new secret key」
3
複製 Key（格式：sk-...）—— 需先儲值
🧠
Anthropic Claude
業界最強推理能力
1
前往 console.anthropic.com/settings/keys
2
點「Create Key」
3
複製 Key（格式：sk-ant-...）
✈️ STEP 2

Telegram Bot 設定

你需要一個屬於自己的 Telegram Bot，大閘蟹才能在 Telegram 裡跟你對話。

1

在 Telegram 搜尋 @BotFather（官方認證藍勾）

2

輸入指令 /newbot

3

依提示為 Bot 取名（例如：My DaZhaxie Bot）

4

再取一個 username（必須以 bot 結尾，例如：my_dazhaxi_bot）

5

BotFather 會給你一串 Token（格式：1234567890:ABCdefGHIjklMNO...）

6

複製並保存這個 Token

⚠️ 重要提醒
  • Token 是你的 Bot 密鑰，請勿分享給他人
  • 一個 Token 對應一個 Bot，建議每人獨立一個
  • Token 不小心洩露？到 BotFather 輸入 /revoke 重新生成
